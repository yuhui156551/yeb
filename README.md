# yeb
## 前后端分离办公项目

### 技术栈

  SpringBoot2、MybatisPlus、Redis、SpringSecurity、JWT、Kaptcha、EasyPOI、AspectJ、MySQL、Lombok、Swagger2、RabbitMQ、Vue。
  
### 实现
  
  主要实现功能：登录校验、职位管理、职称管理...、权限组、导入导出员工数据等。
